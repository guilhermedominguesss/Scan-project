Nome do projeto: scan-de-crescimento-pro-replit

Objetivo: criar uma SPA React + Tailwind responsiva (mobile-first) com UM QUIZ COMPLETO chamado “Scan de Crescimento Profissional”, incluindo Dashboard interno, LP final, animações premium (framer-motion opcional), barra de progresso, tracking de eventos, persistência local e copy altamente persuasiva. Projeto completo 100% frontend — sem backend.

IMPORTANTE — CAPTURA ANTECIPADA (NOVO)
• Ao clicar em “Iniciar o Scan” o usuário deve primeiro preencher **3 campos obrigatórios** antes de iniciar as perguntas:
  1. Nome completo (input text)  
  2. Nome da empresa / estúdio (input text)  
  3. WhatsApp (input com máscara BR (00) 00000-0000)
• Microcopy abaixo: “Preencha esses dados para salvar sua análise caso pare no meio do processo. Leva < 60s no total.”  
• O app deve salvar imediatamente esses 3 campos no localStorage assim que preenchidos (evento analytics: lead_captured_draft).  
• Se o usuário abandonar posteriormente, você já terá os dados para contato.  
• Se o usuário retornar, perguntar “Deseja continuar do ponto X?” e permitir retomar.

Quiz possui:
• Barra de progresso (Passo X/Y) com animação dourada (#CDA580 / #9D6135)  
• Auto-save a cada resposta (localStorage + api.saveDraft com debounce 800ms)  
• Recuperação de sessão (continue do ponto anterior)  
• Animações suaves nas transições (framer-motion opcional)  
• Persistência completa  
• Tracking de eventos definidos abaixo

Dashboard interno:
• Rota protegida com senha guilherme2025 (modal de prompt no front)  
• Tabela completa de leads (ver seção Dashboard mais abaixo)  
• Gráficos SVG simples (funil, abandono por etapa)  
• Export CSV  
• Filtros por data e status  
• Reenvio manual de análises e abertura de conversa via wa.me link

Antes de sair da página:
• beforeunload → registrar quiz_abandoned se não finalizou (lastStep salvo)

ErrorBoundary ativo cobrindo toda a aplicação.

Tela 00 — PRIMEIRA CAPTURA (antes de qualquer pergunta)
Título: “Antes de começar — precisamos de um dado rápido”

• Nome da empresa / estúdio (obrigatório)
• WhatsApp (máscara BR) (obrigatório)

Título: “Para começar sua análise, me conte: qual é exatamente o seu negócio?”
